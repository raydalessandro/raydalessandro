## Progetti Rappresentativi

**[La Famiglia](https://github.com/raydalessandro/la-famiglia)** — PWA familiare in produzione. Next.js 15 + auth custom + Supabase. Workflow maturo: branch → preview → test unit ed E2E → PR → merge. Mai nessuna modifica diretta in produzione.

**[Graph Spec Pipeline](https://github.com/raydalessandro/graph-spec-pipeline)** — Metodologia spec-driven per sviluppo multi-agente. Grafo come fonte di verità, agenti separati per spec/code/review. Open source MIT.

**[EAR Claude Telegram Bot](https://github.com/raydalessandro/ear-claude-telegram-bot)** — Accesso remoto a Claude Code via Telegram. TypeScript + Bun + Whisper locale. 624 test. Open source MIT.

**[67_ENT](https://github.com/raydalessandro/67_ENT)** — Piattaforma Next.js + Supabase per etichetta musicale. Workflow approvazione contenuti, AI chat per artista, analytics Instagram. 181 test Vitest.

**[Grafo_own](https://github.com/raydalessandro/Grafo_own)** — Studio etnografico tradotto in sistema multi-agente (6 agenti orchestrati). Visualizzazione D3 force-directed di organizzazione AI-native.

**[Conflict Resolver](https://github.com/raydalessandro/conflict-resolver)** — Ontologia EAR a 72 simboli come motore d'inferenza TypeScript. 12 archetipi, 8 pattern di conflitto. Privacy-first, zero backend.

**[Isola dei Tre Venti](https://github.com/raydalessandro/isola_i3v_visual)** — Saga 12 storie illustrate per bambini. Pipeline: canone come grafo (116 entità), brieffer Python, cartografia GeoJSON. Precedente iterazione: 1 titolo pubblicato Amazon KU, 68 recensioni, 4 stelle.
